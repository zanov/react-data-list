# react-data-list

SPA simple application for data listing and details viewing

## Pre-requests

To run the app locally you need to have installed:

1. Google Chrome
1. Node.js
1. NPM
1. Git

## Running the application

1. Clone the project https://github.com/zanov/react-data-list.git.
1. cd react-data-list
1. npm install
1. npm start

The Application should be loaded directly in Google Chrome http://localhost:3030. Automatically starts Express.js server as well.

## Running the tests

1. cd react-data-list
1. npm test
